largest of the numbers that were input.

Scenario 3:

Create an algorithm that reads an arbitrary number of integers and then
returns their arithmetic average.

Script:

set a variable that is the sum of integers equal to zero
set counter of integers to zero

loop 
	read an integer 
	add integer to the sum of integers
	increase counter by one
	stop when there are no more unread integers
	
then
	divide the sum of integers by the counter to calculate average
	
return the average



Scenario 4:

Create an algorithm that reads a grocery list and prints out the products
(in alphabetical order) that are still left to buy.

Script:

create three lists
one that is a grocery list and set to zero
one that is of already purchased items and set to zero
one that is of items still left to buy and set to zero

loop
	read a grocery item name
	add the item to grocery list
	stop when there are no more unread grocery items
	
loop
	read grocery list
	add items that have been purchased already to already purchased list
	stop when there are no more unread items
	
loop
	read grocery list and already purchased item list
	add item from grocery list that is not in the already purchased list to the list of items still left to buy
	stop when there are no more groceries left to add to items still left to buy
	
sort list of grocery items still left to buy into alphabetical order
	
print the list of the items still left to buy
